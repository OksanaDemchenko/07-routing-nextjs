# Next.js Routing Project

This is a Next.js and TypeScript web application demonstrating dynamic routing and navigation.  
The project is deployed with Vercel and shows routes, pages, and basic navigation using Next.js features.

## 🧰 Technologies

- Next.js
- React
- TypeScript
- Vercel (deployment)
- CSS / Tailwind (if used)
- Git & GitHub

## 🚀 Features

- Dynamic routing in Next.js
- Multiple pages with links
- Navigation between routes
- Server‑side rendering (SSR / SSG where applicable)
- Responsive layout
